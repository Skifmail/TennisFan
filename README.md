# TennisFan - Tennis Tournament Platform

Django-проект, клон tennison.ru — платформы любительских теннисных турниров.

## Быстрый старт

```bash
cd /home/skifmail/Projects/TennisFan
source venv/bin/activate
python manage.py runserver
```

Откройте http://127.0.0.1:8000/

## Структура проекта

```
tennison/
├── config/              # Настройки Django
├── apps/
│   ├── users/           # Пользователи и игроки
│   ├── tournaments/     # Турниры, матчи, рейтинги
│   ├── courts/          # Теннисные корты
│   ├── sparring/        # Поиск спарринг-партнёров
│   ├── training/        # Тренировки для взрослых
│   ├── content/         # Новости, галереи, страницы
│   └── comments/        # Комментарии
├── templates/           # HTML-шаблоны
├── static/              # CSS, JS, изображения
└── media/               # Загруженные файлы
```

## Возможности админ-панели

- **Пользователи**: создание, редактирование, верификация игроков
- **Турниры**: создание турниров, добавление матчей, ввод результатов
- **Корты**: управление каталогом площадок
- **Тренировки**: тренеры, программы, записи на занятия
- **Контент**: новости, фотогалереи, статические страницы
- **Комментарии**: модерация отзывов

## Технологии

- Python 3.12
- Django 6.0
- SQLite
- Vanilla CSS (dark theme)
